# Digital Crack Gauge

**Measure concrete crack widths from photographs — fast, repeatable, no specialist equipment.**

---

## What it does

Concrete crack width is a key indicator of structural condition. Digital Crack Gauge estimates crack widths directly from photographs taken in the field, turning an ordinary image into a calibrated measurement.

Upload a photo, set the scale, draw a line across the crack — get a width.

Results can be saved as annotated images for reporting and record-keeping.

---

## Access

The tool is access-controlled. A 6-digit code is required to open it. Codes are issued directly — contact the author if you are interested in access or licensing.

---

## Interface tiers

Two tiers are available, unlocked by the access code:

- **Lite** — streamlined interface for straightforward measurements
- **Pro** — full feature set including advanced calibration and result controls

---

## Technical notes

- Runs entirely in the browser — no installation, no server, no data uploaded
- Single file, no dependencies
- Tested on desktop and mobile (Chrome, Firefox, Safari)
